# Simple Calculator 🔢

This is a simple calculator web project built using **HTML**, **CSS**, and **JavaScript**.  
It performs basic arithmetic operations like:

- Addition ➕  
- Subtraction ➖  
- Multiplication ✖️  
- Division ➗

- Instant result display
- Clean and responsive design
